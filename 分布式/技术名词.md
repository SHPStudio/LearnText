**什么叫幂等性**

幂等性在数学中的表示是f(x)=f(f(x))

接口的幂等性是表示，外部接口的调用不会影响我的内部系统。
也就是说幂等接口会对内部系统进行保护，保证外部接口调用的时候不会影响内部系统。

比如Cache实现

     value getValue(key){
     value = getValueFromCache(key);
     if( value == null ){
         value = readFromPersistence(key);
         saveValueIntoCache(key,value);
     }
 
     return value;
